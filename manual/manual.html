Introduction

Blizzard

Blizzard is a simple calculator capable of adding or subtracting two 7-bit integers. It was built in Mincraft out of a huge amount of redstone logic gates - the binary to binary coded decimal converters making up the bulk of the it! Infact, the white section is where the actual calculation is performed. The outputs are pistion-powered 7-segment displays.

Blizzard 4

Blizzard 4 is a 16-bit computer capable of advanced arithmetic (through software), logical operations and basic graphics via its 128x128 display. It has a very small instructon set that is mainly designed to manupulate data. The actual computation is done by the registers that the data can be moved into.

Architecture

Blizzard 4 has a modified von Neumann architecture. Dual address buses take the place of a control bus because this architecture supports the direct copying of data, i.e one instruction could copy one word in RAM to another adress in RAM. This is would be accomplished by:
  • putting the address of the word to be copied on the read bus, which would cause the relevant device to put the corresponding data onto the data bus.
  • putting the address of the place it is to be copied to on the write bus, which would cause the relevant device to store whatever is on the data bus at the specified location.
